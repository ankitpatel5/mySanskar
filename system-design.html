Service Worker
Caches the app shell so it loads instantly from the home screen, even offline. Bumping the version number forces all devices to refresh.

Key Design Decisions
No backend server
All logic runs in the browser. External APIs (Firebase, Drive, Gemini) are called directly from the client with restricted API keys.
Stories pre-bundled
All 256 stories and their translations ship as static JS files. Zero load-time latency — no API call needed to browse or read stories.
Music from Google Drive
Adding new kirtan is as simple as dropping an MP3 into a Drive folder. No CMS, no upload pipeline needed.
Offline-capable PWA
The service worker caches the app shell. Stories and UI work offline. Music streams on-demand (requires connection).
