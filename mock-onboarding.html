Implementation Notes
Trigger
Check localStorage.getItem('drift.onboardingDone') on app load. If missing → show onboarding. Set it to '1' when the user reaches the final screen.
Coach marks
Track seen tabs with drift.seenTabs (JSON array). On first switch to Music or Stories, show the tooltip. Tap anywhere to dismiss and mark it seen.
Skip behaviour
"Skip for now" on the profile screen skips straight to Screen 4 (All set). Profile can be added later via Settings — SOTD shows the "Add your child → Settings" prompt in the meantime.
Guest vs signed-in
Onboarding fires for both. For guests, the profile step still works (saves to localStorage). If they later sign in, existing sync logic uploads it to Firestore.
